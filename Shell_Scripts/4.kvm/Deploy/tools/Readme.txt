#																	    #
# 概述1：																    #
#       - KVM虚拟机是全称是 < Kernel Base Virtual Mechine >									      	    #
#       - 将kernel改造成了一个虚拟化环境vmm， 												    #
#	- 每个虚拟机都是kernel的一个进程												    #
#       - 每个virtual cpu 都是该进程组下的一个线程											    #
#	- 其中，Qemu 模块模拟虚拟硬件													    #
#       - 所有模拟硬件的指令被qemu截获送至kernel层进行处理，再由kernel送给cpu等硬件进行指令计算，最后反馈回qemu和他的虚拟硬件。		    #
#       - 也就是说，kernel依然管理着整个宿主机的硬件等等										    #
#																    	    #
# 概述2：																    #
#       - kvm虚拟机的安装，													            #
#	  - 主要由网络-磁盘-和安装后虚拟机对应的配置文件组成 										    #
#	  - kvm虚拟机的创建命令virt-install，大致重要的几点为 										    #
#	    - 创建磁盘                                                              							    # 
#   	    - 设定虚拟机配置参数（cpu，rom）												    #
#           - 指定iso镜像，指定虚机磁盘映像												    #
#           - 指定连接的网桥													    	    #
#           - 指定vnc的端口等等														    #
#	    - 虚机创建脚本也基于以上思路												    #
#############################################################################################################################################



- 至于可交互的操作  
    virsh -选项  虚拟号/虚拟名  
    virsh 是libvirt 组件提供的一个虚拟机管理指令，可以使用其对kvm虚拟机生命周期等进行方便快捷的管理。
---------------------------------------------------------------------------------------------------------------------------------------------------------------
|    virsh start x                   启动名字为x的非活动虚拟机  											      |
|    virsh list   --all              列出虚拟机      													      |
|    virsh shutdown x                正常关闭虚拟机  													      |
|    virsh destroy x                 强制关闭虚拟机  													      |
|    virsh undefine x                删除虚拟机  													      |
|    virsh dominfo x                 显示虚拟机的基本信息  											      	      | 
|    virsh domname 2                 显示id号为2的虚拟机名  												      | 
|    virsh domid x                   显示虚拟机id号  													      | 
|    virsh domuuid x                 显示虚拟机的uuid  													      |
|    virsh domstate x                显示虚拟机的当前状态  												      | 
|    virsh dumpxml x                 显示虚拟机的当前配置文件（可能和定义虚拟机时的配置不同，因为当虚拟机启动时，需要给虚拟机 分配id号、uuid、vnc端口号等等） |  
|    virsh setmem x 512000           给不活动虚拟机设置内存大小  											      |
|    virsh edit x                    编辑配置文件（一般是在刚定义完虚拟机之后)										      |
---------------------------------------------------------------------------------------------------------------------------------------------------------------
